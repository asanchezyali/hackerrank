# HackerRank Interview Preparation Kit - Python Solutions

This repository contains Python solutions for problems from the [HackerRank Interview Preparation Kit](https://www.hackerrank.com/interview/interview-preparation-kit).

## Project Structure

```
interview_preparation_kit/
├── arrays/
├── dictionaries_and_hashmaps/
├── general/
├── sorting/
└── string_manipulation/
```

## Requirements

- Python >= 3.12

## Setup

This project uses [uv](https://docs.astral.sh/uv/) as the package manager.

```bash
# Install uv (if not already installed)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Sync dependencies
uv sync

# Run a solution
uv run python interview_preparation_kit/arrays/problem_1_2D_array_DS.py
```
